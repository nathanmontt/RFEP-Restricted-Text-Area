<h1 id="about">Sobre o Projeto</h1>

Como o nome do repositório menciona, **RFEP-Restricted-Text-Area** é um projeto de Frontend, trago pelo site <a href="https://roadmap.sh/">roadmap.sh</a>, onde o objetivo é introduzir o estudo de **_Javascript_**. Mais especificamente com _**DOM**_, criando uma área de texto restrita.

<h1 id="abilities"> Habilidades Necessárias </h1>

* HTML
* CSS
* Javascript
